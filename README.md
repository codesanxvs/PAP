# Flash Cabeleireiro

## Descrição
Aplicação web desenvolvida para a gestão de um cabeleireiro, permitindo efetuar marcações, consultar serviços e gerir conteúdos através de uma área administrativa.

## Tecnologias Utilizadas
- HTML
- CSS
- JavaScript
- Node.js
- API do Notion

## Instalação
npm install

## Execução
node server_notion_js.js

## Funcionalidades
- Marcação de serviços
- Gestão de clientes
- Loja online
- Painel de administração
